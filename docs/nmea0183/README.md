# NMEA 0183 API Index

This index lists all supported NMEA 0183 message classes.

| Message | Description |
|---|---|
| [APB](APB.md) | Autopilot sentence with cross-track error and steering guidance. |
| [DBT](DBT.md) | Depth below transducer in feet, meters, and fathoms. |
| [DPT](DPT.md) | Water depth and transducer offset information. |
| [GGA](GGA.md) | GPS fix data including position, quality, and altitude. |
| [GLL](GLL.md) | Geographic latitude/longitude position with status and mode. |
| [GSA](GSA.md) | GNSS DOP values and active satellite IDs. |
| [GSV](GSV.md) | Satellites in view with elevation, azimuth, and SNR. |
| [HDG](HDG.md) | Magnetic heading with deviation and variation. |
| [HDM](HDM.md) | Magnetic heading only. |
| [MTW](MTW.md) | Water temperature measurement. |
| [MWV](MWV.md) | Wind angle and speed with reference and status. |
| [RMA](RMA.md) | Recommended minimum navigation data (legacy variant). |
| [RMB](RMB.md) | Recommended minimum navigation information to a waypoint. |
| [RMC](RMC.md) | Recommended minimum GNSS data (time, position, speed, course). |
| [VHW](VHW.md) | Water-referenced heading and speed. |
| [VLW](VLW.md) | Distance traveled through water/ground since reset and total. |
| [VTG](VTG.md) | Course over ground and speed over ground. |
| [VWR](VWR.md) | Relative wind direction and speed. |
| [XDR](XDR.md) | Transducer measurements (typed sensor readings). |
| [XTE](XTE.md) | Cross-track error and steering direction. |
| [ZDA](ZDA.md) | UTC time and date with local zone offset. |
