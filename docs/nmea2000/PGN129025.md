# PGN129025 API

Class `nmea2000.PGN129025` (Python) / `nmealib::nmea2000::PGN129025` (C++).

## PGN Overview
CANboat: PGN 129025 - Position, Rapid Update. Reports rapidly updated latitude and longitude.

## Creation
### C++
```cpp
// Direct constructor:
nmealib::nmea2000::PGN129025 msg = nmealib::nmea2000::PGN129025(Latitude latitude,
              Longitude longitude
    );

// Factory method from raw data:
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```

### Python
```python
# Direct constructor:
msg = nmealib.nmea2000.PGN129025(/* fields */)

# Factory method from raw data:
msg = nmealib.nmea2000.Nmea2000Factory.create(raw_can_frame)
```
