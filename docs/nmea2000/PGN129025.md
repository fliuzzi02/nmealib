# PGN129025 API

Class `nmea2000.PGN129025` (Python) / `nmealib::nmea2000::PGN129025` (C++).

## PGN Overview
CANboat: PGN 129025 - Position, Rapid Update. Reports rapidly updated latitude and longitude.

## Creation
- Python direct:
```python
msg = nmealib.nmea2000.PGN129025(<fields...>)
```
- C++ direct:
```cpp
nmealib::nmea2000::PGN129025 msg(/* fields */);
```
- Python factory:
```python
msg = nmealib.nmea2000.Nmea2000Factory.create(raw_can_frame)
```
- C++ factory:
```cpp
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```

## Public Methods

### `get_latitude` / `getLatitude`

```cpp
Latitude getLatitude() const noexcept;
```

Returns the latitude value parsed from this PGN.

### `get_longitude` / `getLongitude`

```cpp
Longitude getLongitude() const noexcept;
```

Returns the longitude value parsed from this PGN.

### `get_string_content` / `getStringContent`

```cpp
std::string getStringContent(bool verbose) const noexcept override;
```

Returns a human-readable representation of the PGN content.
