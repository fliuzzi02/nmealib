# PGN129026 API

Class `nmea2000.PGN129026` (Python) / `nmealib::nmea2000::PGN129026` (C++).

## PGN Overview
CANboat: PGN 129026 - COG & SOG, Rapid Update. Reports course over ground and speed over ground.

## Creation
### C++
```cpp
// Direct constructors:
nmealib::nmea2000::PGN129026 msg = nmealib::nmea2000::PGN129026(uint8_t sequenceId,
              HalfByte cogReference,
              Angle cog,
              Speed sog
    );

nmealib::nmea2000::PGN129026 msg2 = nmealib::nmea2000::PGN129026(uint8_t sequenceId,
              HalfByte cogReference,
              Byte reserved,
              Angle cog,
              Speed sog,
              Byte reserved2,
              Byte reserved3
    );

// Factory method from raw data:
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```

### Python
```python

msg = nmealib.nmea2000.PGN129026(/* fields */)
```
