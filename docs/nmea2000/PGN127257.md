# PGN127257 API

Class `nmea2000.PGN127257` (Python) / `nmealib::nmea2000::PGN127257` (C++).

## PGN Overview
CANboat: PGN 127257 - Attitude. Reports yaw, pitch, and roll attitude angles.

## Creation
### C++
```cpp
// Direct constructor:
nmealib::nmea2000::PGN127257 msg = nmealib::nmea2000::PGN127257(uint8_t sequenceId,
              SignedAngle yaw,
              SignedAngle pitch,
              SignedAngle roll,
              Byte reserved
    );

// Factory method from raw data:
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```

### Python
