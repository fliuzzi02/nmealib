# PGN127251 API

Class `nmea2000.PGN127251` (Python) / `nmealib::nmea2000::PGN127251` (C++).

## PGN Overview
CANboat: PGN 127251 - Rate of Turn. Reports turn-rate around the vertical axis.

## Creation
### C++
```cpp
// Direct constructor:

              AngularRate rate,
              Byte reserved1,
              Byte reserved2,
              Byte reserved3
    );

// Factory method from raw data:
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```
