# PGN127251 API

Class `nmea2000.PGN127251` (Python) / `nmealib::nmea2000::PGN127251` (C++).

## PGN Overview
CANboat: PGN 127251 - Rate of Turn. Reports turn-rate around the vertical axis.

## Creation
- Python direct:
```python
msg = nmealib.nmea2000.PGN127251(<fields...>)
```
- C++ direct:
```cpp
nmealib::nmea2000::PGN127251 msg(/* fields */);
```
- Python factory:
```python
msg = nmealib.nmea2000.Nmea2000Factory.create(raw_can_frame)
```
- C++ factory:
```cpp
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```
