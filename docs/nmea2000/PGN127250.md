# PGN127250 API

Class `nmea2000.PGN127250` (Python) / `nmealib::nmea2000::PGN127250` (C++).

## PGN Overview
CANboat: PGN 127250 - Vessel Heading. Reports vessel heading with magnetic deviation and variation components.

## Creation
- Python direct:
```python
msg = nmealib.nmea2000.PGN127250(<fields...>)
```
- C++ direct:
```cpp
nmealib::nmea2000::PGN127250 msg(/* fields */);
```
- Python factory:
```python
msg = nmealib.nmea2000.Nmea2000Factory.create(raw_can_frame)
```
- C++ factory:
```cpp
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```
