# PGN128259 API

Class `nmea2000.PGN128259` (Python) / `nmealib::nmea2000::PGN128259` (C++).

## PGN Overview
CANboat: PGN 128259 - Speed. Reports water-referenced and ground-referenced speed values.

## Creation
### C++
```cpp
// Direct constructor:
nmealib::nmea2000::PGN128259 msg = nmealib::nmea2000::PGN128259(uint8_t sequenceId,
              Speed speedWaterReferenced,
              Speed speedGroundReferenced,
              Byte speedWaterReferencedType,
              HalfByte speedDirection,
              Byte reserved1,
              HalfByte reserved2
    );

// Factory method from raw data:
auto msg = nmealib::nmea2000::Nmea2000Factory::create(rawCanFrame);
```
