# PGN130306 API

Class `nmea2000.PGN130306` (Python) / `nmealib::nmea2000::PGN130306` (C++).

## PGN Overview
CANboat: PGN 130306 - Wind Data. Reports wind speed, wind direction, and wind reference.

## Creation
### C++
```cpp
// Direct constructors:
nmealib::nmea2000::PGN130306 msg = nmealib::nmea2000::PGN130306(uint8_t sequenceId,
              Speed windSpeed,
              Angle windDirection,
              HalfByte windReference
    );

nmealib::nmea2000::PGN130306 msg2 = nmealib::nmea2000::PGN130306(uint8_t sequenceId,
              Speed windSpeed,
              Angle windDirection,
              HalfByte windReference,

// Factory method from raw data:
```
