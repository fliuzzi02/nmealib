# PGN127245 API

Class `nmea2000.PGN127245` (Python) / `nmealib::nmea2000::PGN127245` (C++).

## PGN Overview
CANboat: PGN 127245 - Rudder. Reports rudder instance, direction, ordered angle, and actual rudder position.
